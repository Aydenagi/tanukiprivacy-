# LLM Integration

LLMs are optional and advisory.

---

## Allowed Uses

- Explaining alerts
- Summarizing event sequences
- Assisting manual investigation

---

## Forbidden Uses

- Risk scoring
- Automated actions
- Policy decisions
- Threshold tuning

---

## Trust Boundary

All decisions occur before any LLM interaction.

If all LLM code is removed, Tanuki remains fully functional.
